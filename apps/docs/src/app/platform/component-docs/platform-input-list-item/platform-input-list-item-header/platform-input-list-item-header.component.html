Input List Item

The input list item is the simplest list item. It shows content in the form of labels and text. This component makes use of list-item directive present in fundamental-ngx/core and has additional enhancements on top of it.

Platform specific enhancements:

  • List-item is implemented as Angular component to hide complexity of dealing with HTML in order to deliver high-order component.
  • Alignment of sub elements title, footer, group header are abstracted.
  • Enhanced support for accessibility which covers access by keyboard, mouse and touch.
  • Enhanced compatibility with assistive technology (screen readers).
