Carosuel With No Control Carousel with no control where slides will only displayed automtically after 3000 time interval with default image alignment. [isControls]="'false'" and [animation]="'slide'" options are used to achive this. Carosuel With Control Carousel with control where slides display can be controlled by using next and previous buttons. [isControls]="'true'" it is default value. [animation]="'slide'" options are used to achieve this. Carosuel With Top Indicator Carousel with top Indicator control where slides display can be controlled by using next and previous buttons. [isControls]="'true'" it is default value. [animation]="'slide'" and set [indicatorPosition]="'value''" to place the indicator as per our requirement. Carosuel With previous and next navigations along with Indicators Carousel with previous and next navigations along with Indicators where slides display can be controlled by using next and previous buttons. [isControls]="'true'" it is default value. [animation]="'slide'" and set [indicatorPosition]="'value'" and set [inMiddle]="false|true" true will make the previous and next navigations in middle of the slides and false will display it along with page indicators. Carosuel With Defined intervals Interval attribute will help to give the defined intervals between slide display. [interval]="'value'" required to be set. Carosuel With Caption Each slide will have some caption. [caption]="'value'" required to be set in "'fdp-carousel-item'" tag. Videos in Carosuel We can have vedios also in carousel. Need to give the vedio source details in fdp-carosuel-item. Carosuel With Number Indicator When number of slides reached more than 8 then dots will be converted into numbers as shown in example. Error handling in Carosuel Validate a required senario and update the [error]="'true'" Loading indicator in Carosuel Validate a required senario and update the [loading]="'true'" to display loading indicator
